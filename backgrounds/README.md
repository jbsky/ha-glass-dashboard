## Image Guidelines
- Resolution: 1920x1080 minimum (landscape)
- Format: JPEG, optimized for web (~200KB each)
- Style: atmospheric, slightly dark for glass readability
- Tip: Use photos from your local area for a personal touch

## Sourcing Images
We recommend using [Unsplash](https://unsplash.com) with a search like:
- "provence sunny landscape"
- "mediterranean cloudy sky"
- "night sky stars"
- "rain window"

All Unsplash photos are free to use (Unsplash License).
